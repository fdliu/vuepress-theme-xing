## 快速上手

```bash
# clone the project
git clone https://github.com/fdliu/vuepress-theme-xing.git

# enter the project directory
cd vuepress-theme-xing

# install dependency
npm install # or yarn install

# develop
npm run dev # or yarn dev
```
